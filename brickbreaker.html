Tilt Brickbreaker

Created the popular game Brickbreaker in Python and controlled the paddle using an accelerometer in C.

This was a two-person final project for my Embedded Systems class (ECE3140), which took input from the accelerator embedded on our FRDM-KL46Z MPU board, processed and filtered the values, and then used serial communication to communicate with the Python program to control the paddle on the Python GUI.
